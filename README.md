# Análisis de Datos Climáticos

## Integrantes
- Natalia Micaela Segura

## Escenario elegido
Análisis de datos climáticos globales utilizando Python y Google Colab.

## Dataset utilizado
Global Temperature Dataset: https://datahub.io/core/global-temp

## Funcionalidades
- Lectura de datos CSV
- Cálculo de indicadores climáticos
- Visualización gráfica
- Exportación de resultados

## Estructura del proyecto
- datos/: dataset utilizada
- scripts/: scripts de análisis
- resultados/: gráficos y resultados generados

## Ejecución
Ejecutar el script principal en Google Colab
